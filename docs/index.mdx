# Путёвка в жизнь

«Путёвка в жизнь» — Django-платформа для сопровождения абитуриентов: от сбора данных и проверки анкет до подготовки документов, отслеживания прогресса и коммуникации с командой проекта.

## Основные возможности

- Личный кабинет участника и регистрационные сценарии.
- Staff-панели для проверки профиля, документов, учебных данных, интервью и тестирования.
- Генерация документов и управление шаблонами.
- Интеграции с Telegram, почтой и сервисом звонков.
- Отдельный AI-сервис для транскрибации, заполнения форм и помощи с проверкой.

## Структура репозитория

| Путь | Назначение |
| --- | --- |
| `Putevka/` | Настройки Django-проекта, URL, ASGI/WSGI entrypoints. |
| `core/` | Общие views, сервисы, шаблоны, уведомления и интеграция AI-задач. |
| `documents/` | Модели, формы, views и сервисы генерации документов. |
| `my_study/` | Учебные формы, fixtures и views. |
| `review_by_tutor/` | Staff-сценарии проверки и инструменты для интервью. |
| `scholar_form/` | Шаги анкеты участника и связанные сервисы. |
| `subscriber/` | Подписки и сценарии приглашений. |
| `ai_service/` | Отдельный worker-сервис для AI-задач. |
| `.github/workflows/` | CI, деплой и публикация документации. |

## Технологии

- Django
- PostgreSQL или SQLite для локальной разработки
- Docker Compose
- Bootstrap-шаблоны
- GitHub Actions
- Mintlify для документации
